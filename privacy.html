🔒 隐私政策

最后更新：2026年

📌 一句话总结

鱼隐·离线钓场 不收集任何个人信息。 游戏数据使用 Core Data 存储在设备本地；若你登录了 iCloud，数据可通过 Apple 的 CloudKit 自动同步到你的其他 Apple 设备。开发者无法访问你的 iCloud 数据——这是 Apple 架构的保证。

1. 信息收集

✅ 我们不收集的信息

  • ✓ 不收集个人身份信息（姓名、邮箱、电话等）
  • ✓ 不收集位置信息
  • ✓ 不收集设备标识符（IDFA、IMEI等）
  • ✓ 不收集使用数据或分析数据
  • ✓ 不收集照片、通讯录、麦克风等敏感信息
  • ✓ 不需要创建账户或登录

⚠️ 我们可能使用的系统功能

  • ✓ Core Motion（陀螺仪） — 仅用于体感抛竿操作，数据在设备内实时处理，不会被记录或上传
  • ✓ Core Haptics（触觉反馈） — 本地触觉引擎，纯设备内响应
  • ✓ StoreKit（内购） — Apple 提供的标准支付API，应用本身无法获取你的支付信息

2. 数据存储

所有游戏数据（包括钓鱼记录、收藏、设置、排行榜分数）均使用 Apple 的 Core Data 框架存储在设备本地。

☁️ iCloud 跨设备同步（可选）

本应用使用 NSPersistentCloudKitContainer。当你的设备登录 iCloud 时，游戏数据（图鉴收藏、称号进度、游戏记录等）会通过 Apple 的 CloudKit 自动同步到你的其他 Apple 设备。

  • ✓ 这是 Apple 系统级同步，由 Apple 的基础设施控制——开发者无法访问你的 CloudKit 数据
  • ✓ 数据在传输和存储时均受到 Apple 的标准加密保护
  • ✓ 同步是自动的——只要设备登录了 iCloud，游戏数据就会同步到其他 Apple 设备
  • ✓ 应用内没有同步开关，无法单独关闭同步
  • ✓ 如需删除已同步的 iCloud 数据，可通过以下两种方式：
  • ① iOS 设置 › Apple ID › iCloud › 本应用 中进行删除
  • ② 应用内「设置」页面的「删除 iCloud 数据」按钮
  • ✓ 如果设备未登录 iCloud，数据完全存在于设备本地，不上传到任何位置

💾 本地存储

  • ✓ 数据存储在设备内部存储中，不备份到第三方云服务
  • ✓ 可通过 iOS 系统设置清除应用数据
  • ✓ 卸载应用将同时删除所有本地数据
  • ✗ 不存在开发者服务器——我们没有任何服务器

3. 数据传输与网络安全

本应用代码中不包含任何主动网络请求逻辑，应用本身不向任何外部服务器发送数据。

☁️ CloudKit 同步层（Apple 系统行为）

当 iCloud 处于开启状态时，iOS 系统内置的 Core Data + CloudKit 层会通过 Apple 的 iCloud API 在设备间同步游戏数据。这一行为：

  • ✓ 由 Apple 的基础设施（而非应用代码）控制
  • ✓ 使用 Apple 标准加密，传输和存储均受到保护
  • ✓ 开发者无法读取或访问用户的 CloudKit 数据
  • ✓ 可在 iOS 设置中删除已同步的 iCloud 数据，无法单独关闭同步

📵 未登录 iCloud 时

  • ✓ 数据仅保存在本地，不会同步到任何位置
  • ✓ 应用本身不发起任何网络连接
  • ✓ 飞机模式、离线环境下均可正常使用

4. 第三方服务

本应用不集成任何第三方 SDK（软件开发工具包），包括但不限于：

  • ✗ 无广告 SDK
  • ✗ 无分析 SDK（如 Google Analytics、Firebase）
  • ✗ 无社交登录 SDK
  • ✗ 无崩溃报告 SDK（如 Crashlytics）
  • ✗ 无推送通知服务

唯一涉及 Apple 系统的外部交互是 StoreKit（应用内购买），该过程由 Apple 直接处理，应用本身无法获取你的支付信息。

5. 儿童隐私

本应用适合所有年龄段用户。由于不收集任何个人信息，不存在收集儿童数据的问题。我们的应用不包含任何不适合儿童的内容。

6. 隐私政策的变更

如果我们更新本隐私政策，将在本页面上发布更新版本。由于应用完全离线，不需要通过推送通知告知用户。

🔒 Privacy Policy

Last updated: 2026

📌 In a Nutshell

YuYin · Offline Fishing does not collect any personal information. Game data is stored locally on your device using Core Data. If you are signed in to iCloud, data can automatically sync to your other Apple devices via CloudKit — a system-level feature managed entirely by Apple. Developers cannot access your iCloud data; that is guaranteed by Apple's architecture.

1. Information Collection

✅ What We Do NOT Collect

  • ✓ No personally identifiable information (name, email, phone, etc.)
  • ✓ No location data
  • ✓ No device identifiers (IDFA, IMEI, etc.)
  • ✓ No usage analytics or telemetry
  • ✓ No photos, contacts, microphone, or other sensitive data
  • ✓ No account creation or login required

⚠️ System Features We May Use

  • ✓ Core Motion (Gyroscope) — Used exclusively for motion-based casting. All processing happens locally on your device in real time; no data is recorded or transmitted.
  • ✓ Core Haptics — Local haptic feedback engine; purely on-device responses.
  • ✓ StoreKit (In-App Purchases) — Apple's standard payment API. The app itself never has access to your payment information.

2. Data Storage

All game data (catch records, collections, settings, leaderboard scores) is stored locally using Apple's Core Data framework.

☁️ iCloud Cross-Device Sync (Optional)

This app uses NSPersistentCloudKitContainer. When your device is signed in to iCloud, game data (fish collection, titles, progress, etc.) automatically syncs to your other Apple devices via Apple's CloudKit.

  • ✓ This is an Apple system-level sync controlled entirely by Apple's infrastructure — developers cannot access your CloudKit data
  • ✓ All data is encrypted by Apple both in transit and at rest
  • ✓ Sync is automatic — when your device is signed in to iCloud, game data syncs across Apple devices automatically
  • ✓ There is no in-app toggle to disable sync
  • ✓ To delete synced iCloud data, you can:
  • ① Go to iOS Settings › Apple ID › iCloud › this app and delete data
  • ② Use the "Delete iCloud Data" button in the app's Settings page
  • ✓ If the device is not signed in to iCloud, all data stays entirely on the device

💾 Local Storage

  • ✓ Data resides in device internal storage; no third-party cloud backup
  • ✓ Can be cleared via iOS system settings
  • ✓ Uninstalling the app removes all local data
  • ✗ No developer-owned server exists — we have no servers at all

3. Data Transmission & Network Security

The app code contains zero active networking logic — the app itself sends no data to any external server.

☁️ CloudKit Sync Layer (Apple System Behavior)

When iCloud is enabled, iOS's built-in Core Data + CloudKit layer syncs game data between your devices via Apple's iCloud API. This behavior:

  • ✓ Is controlled by Apple's infrastructure, not the app code
  • ✓ Uses Apple's standard encryption for both transfer and storage
  • ✓ Is inaccessible to the developer — Apple's architecture prevents it
  • ✓ To remove synced iCloud data, go to iOS Settings › Apple ID › iCloud › this app

📵 When Not Signed into iCloud

  • ✓ All data stays local with no sync
  • ✓ The app itself initiates no network connections
  • ✓ Works perfectly in Airplane Mode with no connectivity

4. Third-Party Services

This application does not integrate any third-party SDKs, including but not limited to:

  • ✗ No ad SDKs
  • ✗ No analytics SDKs (Google Analytics, Firebase, etc.)
  • ✗ No social login SDKs
  • ✗ No crash reporting SDKs (Crashlytics, etc.)
  • ✗ No push notification services

The only external interaction with Apple's system is StoreKit (in-app purchases), which is handled entirely by Apple. The app itself cannot access your payment information.

5. Children's Privacy

This application is suitable for users of all ages. Since we do not collect any personal information, there is no issue of collecting data from children. The app contains no content unsuitable for children.

6. Changes to This Policy

If we update this privacy policy, the revised version will be posted on this page. As the app is fully offline, no push notifications are used to communicate changes.

🔒 プライバシーポリシー

最終更新：2026年

📌 一言でいうと

ユイン · オフライン釣りは、個人情報を一切収集しません。 ゲームデータはCore Dataを使用して端末内にローカル保存されます。iCloudにサインインしている場合、データはAppleのCloudKitを通じて他のAppleデバイスに自動同期されます。これはAppleが管理するシステムレベルの機能であり、開発者がお客様のiCloudデータにアクセスすることはできません。

1. 情報収集について

✅ 収集しない情報

  • ✓ 個人を特定できる情報（氏名、メールアドレス、電話番号など）は収集しません
  • ✓ 位置情報は収集しません
  • ✓ 端末識別子（IDFA、IMEIなど）は収集しません
  • ✓ 使用状況データや分析データは収集しません
  • ✓ 写真、連絡先、マイクなどの機密情報は収集しません
  • ✓ アカウント作成やログインは不要です

⚠️ 使用する可能性のあるシステム機能

  • ✓ Core Motion（ジャイロセンサー） — モーション操作によるキャストにのみ使用。すべての処理は端末内でリアルタイムに行われ、データが記録・送信されることはありません。
  • ✓ Core Haptics（触覚フィードバック） — ローカル触覚エンジン。端末内でのみ動作します。
  • ✓ StoreKit（アプリ内課金） — Apple が提供する標準的な決済API。アプリ自体がお客様の支払い情報にアクセスすることはできません。

2. データ保存について

すべてのゲームデータ（釣果記録、コレクション、設定、ランキングスコア）は、Apple の Core Data フレームワークを使用して端末内にローカル保存されます。

☁️ iCloud クロスデバイス同期（任意）

本アプリは NSPersistentCloudKitContainer を使用しています。デバイスがiCloudにサインインしている場合、ゲームデータ（図鑑コレクション、称号、進行状況など）はAppleの CloudKit を通じて他のAppleデバイスに自動同期されます。

  • ✓ これはAppleのインフラストラクチャが管理するシステムレベルの同期であり、開発者はお客様のCloudKitデータにアクセスできません
  • ✓ データの転送・保存はAppleの標準暗号化で保護されます
  • ✓ 同期は自動で行われます——デバイスがiCloudにサインインしていれば、ゲームデータは自動的に他のAppleデバイスに同期されます
  • ✓ アプリ内に同期ON/OFFスイッチはなく、同期を個別に停止することはできません
  • ✓ 同期済みのiCloudデータを削除するには、以下の方法があります：
  • ① iOS設定 › Apple ID › iCloud › 本アプリ からデータを削除
  • ② アプリ内「設定」ページの「iCloudデータを削除」ボタン
  • ✓ デバイスがiCloudにサインインしていない場合、データは完全に端末内にのみ保存されます

💾 ローカルストレージ

  • ✓ データは端末の内部ストレージに保存され、サードパーティのクラウドにはバックアップされません
  • ✓ iOS のシステム設定からアプリデータを消去できます
  • ✓ アプリをアンインストールすると、すべてのローカルデータが削除されます
  • ✗ 開発者所有のサーバーは存在しません — サーバーは一切ありません

3. データ送信とネットワークセキュリティ

本アプリはネットワーク権限を一切必要としません。アプリコードにはネットワーク通信ロジックは含まれていません。そのため：

  • ✓ データ送信は発生しません
  • ✓ ネットワーク攻撃を受ける可能性はありません
  • ✓ 送信途中でのデータ傍受のリスクはありません
  • ✓ 機内モード、オフライン環境でも問題なく動作します

4. サードパーティサービス

本アプリはサードパーティ製 SDK（ソフトウェア開発キット）を一切統合していません。具体的には：

  • ✗ 広告 SDK なし
  • ✗ 分析 SDK（Google Analytics、Firebase など）なし

  • ✗ クラッシュレポート SDK（Crashlytics など）なし
